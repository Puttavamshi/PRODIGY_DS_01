# Task 01 - Data Visualization

## 📌 Internship

Prodigy InfoTech - Data Science Internship

## 🎯 Objective

Create a bar chart or histogram to visualize the distribution of a variable.

## 📊 Project Description

In this task, I analyzed global population data and created a visualization of the top 10 most populated countries in 2024.

## 🛠️ Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## 🔍 Key Learnings

* Data cleaning and preprocessing
* Handling real-world datasets
* Data visualization techniques
* Adding data labels for better readability

## 🌍 Insight

India has surpassed China as the most populous country in 2024.
